    Schema Updates - Copy & Paste Guide

    File: server/db/schema.ts

    Update 1: Add fields to projects table

    Location: Around line 336, after description: text('description'),

    Add these lines: 

        // Simplified architecture: project IS the prototype
    currentVersion: integer('current_version').default(0).notNull(),
    currentCode: text('current_code'),

    Full context (for reference): 

    export const projects = pgTable('projects', {
    id: uuid('id').primaryKey().defaultRandom(),
    userId: uuid('user_id').references(() => users.id, { onDelete: 'cascade' }),
    organizationId: uuid('organization_id').references(() => organizations.id, { onDelete: 'cascade' }),
    name: text('name').notNull(),
    description: text('description'),
    // ADD THESE TWO LINES HERE:
    currentVersion: integer('current_version').default(0).notNull(),
    currentCode: text('current_code'),
    // END OF ADDITIONS
    githubRepo: text('github_repo'),
    githubBranch: text('github_branch').default('main'),
    webhookUrl: text('webhook_url'),
    githubToken: text('github_token'),
    createdAt: timestamp('created_at').defaultNow().notNull(),
    updatedAt: timestamp('updated_at').defaultNow().notNull(),
});

    Update 2: Add relation to projectsRelations

    Location: Around line 382, after prototypes: many(prototypes),

    Add this line: 

        conversationMessages: many(conversationMessages),

    Full context (for reference): 

    export const projectsRelations = relations(projects, ({ one, many }) => ({
    user: one(users, {
        fields: [projects.userId],
        references: [users.id],
    }),
    organization: one(organizations, {
        fields: [projects.organizationId],
        references: [organizations.id],
    }),
    prototypes: many(prototypes),
    // ADD THIS LINE HERE:
    conversationMessages: many(conversationMessages),
    // END OF ADDITION
}));

    Update 3: Add conversationMessagesRelations

    Location: After conversationMessages table definition and type exports (around line 425)

    Add this entire block: 

    export const conversationMessagesRelations = relations(conversationMessages, ({ one }) => ({
    project: one(projects, {
        fields: [conversationMessages.projectId],
        references: [projects.id],
    }),
}));

    Full context (for reference): 

    export type ConversationMessage = typeof conversationMessages.$inferSelect;
export type NewConversationMessage = typeof conversationMessages.$inferInsert;

    After making these changes, verify:

    1. No TypeScript errors: 

      npm run type-check
# or just check in your editor

    2. Schema is valid: 

      npm run db:push

    3. All imports are correct:

    4. Make sure conversationMessages is imported/defined before the relations

    Quick Commands

    # 1. Open the file
code server/db/schema.ts

# 2. Make the 3 updates above

# 3. Save the file

# 4. Run migration
psql $DATABASE_URL < migrations/add_conversation_messages.sql

# 5. Verify
npm run db:push

# 6. Restart dev server
npm run dev

    Troubleshooting

    Error: "conversationMessages is not defined"

    • Make sure the conversationMessages table is defined before the relations
    • It should already be there from the previous commit

    Error: "Cannot find name 'integer'"

    • Make sure integer is imported at the top: 
      import { pgTable, text, timestamp, uuid, jsonb, index, boolean, integer, vector } from 'drizzle-orm/pg-core';

    Error: "Cannot find name 'relations'"

    • Make sure relations is imported: 
      import { relations } from 'drizzle-orm';
