    üîå API Endpoints

    Assistants API (/api/assistants)

    Method Endpoint Description
    GET /api/assistants List all assistants (with filters)
    GET /api/assistants/:id Get assistant with agents
    POST /api/assistants Create new assistant
    PATCH /api/assistants/:id Update assistant
    DELETE /api/assistants/:id Delete assistant
    POST /api/assistants/:id/duplicate Duplicate assistant
    GET /api/assistants/meta/categories Get all categories

    Query Parameters: - userId - Filter by user - category - Filter by category - isPublic - Show only public assistants - search - Search by name/description

    Agents API (/api/agents)

    Method Endpoint Description
    GET /api/agents List all agents (with filters)
    GET /api/agents/:id Get agent with execution history
    POST /api/agents Create new agent
    PATCH /api/agents/:id Update agent
    DELETE /api/agents/:id Delete agent
    POST /api/agents/:id/execute Execute agent task
    GET /api/agents/:id/executions Get execution history
    PATCH /api/agents/:id/toggle Toggle active status
    GET /api/agents/meta/roles Get available roles

    Query Parameters: - userId - Filter by user - assistantId - Filter by assistant - role - Filter by role - isActive - Filter active agents
