    üé® Design Guidelines

    Screenshot Standards

    Resolution: - Desktop: 1920x1080 or 1440x900 - Mobile: 375x667 (iPhone SE) or 390x844 (iPhone 12) - Tablet: 768x1024 (iPad)

    Format: - PNG for screenshots (lossless) - SVG for diagrams (scalable) - JPEG for photos (if needed)

    Quality: - High DPI (2x for Retina displays) - Clear text (readable at 100%) - No personal data visible - Professional appearance

    Annotations: - Use red arrows/boxes for highlights - Add numbered callouts for steps - Include brief labels - Maintain consistent style

    Diagram Standards

    Tools: - Draw.io (diagrams.net) - Mermaid (markdown diagrams) - Lucidchart - Figma

    Style: - Consistent colors (brand colors) - Clear labels - Logical flow (top-to-bottom or left-to-right) - Legend if needed

    Colors: - Primary: #3B82F6 (blue) - Success: #10B981 (green) - Warning: #F59E0B (orange) - Error: #EF4444 (red) - Neutral: #6B7280 (gray)

    üìù How to Add Screenshots

    Step 1: Capture Screenshots

    Desktop: 1. Open LocalBridge in browser 2. Set browser to 1920x1080 or full screen 3. Navigate to the feature 4. Use screenshot tool (Cmd+Shift+4 on Mac, Win+Shift+S on Windows) 5. Capture the relevant area

    Mobile: 1. Open browser dev tools (F12) 2. Toggle device toolbar (Cmd+Shift+M) 3. Select device (iPhone SE or iPhone 12) 4. Navigate to feature 5. Capture screenshot

    Step 2: Annotate Screenshots

    Using Preview (Mac): 1. Open screenshot in Preview 2. Click markup toolbar 3. Add arrows, boxes, text 4. Save

    Using Paint (Windows): 1. Open screenshot in Paint 2. Use shapes and text tools 3. Add annotations 4. Save as PNG

    Using Online Tools: - Annotely (annotely.com) - Markup.io - CloudApp

    Step 3: Optimize Images

    Using ImageOptim (Mac): 

    # Install
brew install imageoptim-cli

# Optimize
imageoptim docs/assets/screenshots/*.png

    Using TinyPNG (Online): - Visit tinypng.com - Upload images - Download optimized versions

    Step 4: Add to Documentation

    Markdown syntax: 

    ![Login Page](./assets/screenshots/user/login-page.png)
*Figure 1: LocalBridge login page with product showcase*

    With sizing: 

    <img src="./assets/screenshots/user/login-page.png" alt="Login Page" width="800">

    üîÑ Update Process

    When to Update Screenshots

    • Major UI changes - Update affected screenshots
    • New features - Add new screenshots
    • Bug fixes - Update if UI changed
    • Quarterly - Review all screenshots

    Version Control

    Naming convention: 

    feature-name-v1.0.0.png
feature-name-v1.1.0.png

    Git LFS (for large files): 

    # Install Git LFS
git lfs install

# Track image files
git lfs track "docs/assets/**/*.png"
git lfs track "docs/assets/**/*.jpg"

# Commit
git add .gitattributes
git commit -m "Configure Git LFS for images"
