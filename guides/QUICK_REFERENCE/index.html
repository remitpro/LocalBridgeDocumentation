    üöÄ Quick Start

    Development

    # Install dependencies
npm install

# Start development servers (both frontend + backend)
npm run dev

# Type checking
npm run type-check

# Build for production
npm run build

    Database

    # Generate migration
npm run db:generate

# Run migrations
npm run db:migrate

# Open Drizzle Studio
npm run db:studio

    üîë Key Files

    Frontend

    File Purpose
    src/App.tsx Main app with routing
    src/components/auth/LoginPage.tsx Landing/login page
    src/context/AuthContext.tsx Authentication state
    src/lib/analytics.ts Analytics service
    src/lib/api.ts API client

    Backend

    File Purpose
    server/index.ts Server entry point
    server/routes/generate.ts AI generation endpoint
    server/lib/db.ts Database connection
    server/lib/auth.ts Auth middleware
    server/lib/metrics.ts Performance tracking

    Configuration

    File Purpose
    vite.config.ts Vite configuration
    tsconfig.json TypeScript config (frontend)

    üõ†Ô∏è Common Tasks

    Add a New Page

    1. Create component in src/pages/
    2. Add route in src/App.tsx
    3. Add analytics tracking
    4. Update navigation if needed

    Add a New API Endpoint

    1. Create route in server/routes/
    2. Add authentication middleware
    3. Add audit logging
    4. Register in server/index.ts

    Add Analytics Event

    import { analytics } from '@/lib/analytics';

// Track event
analytics.track({
    category: 'Feature',
    action: 'Used',
    label: 'Feature Name',
    metadata: { /* optional */ }
});

// Track button click
analytics.buttonClick('Button Name', 'Page Name');

// Track form submit
analytics.formSubmit('Form Name', success, errorMessage);

    Add Loading Skeleton

    import { SkeletonCard } from '@/components/loading/SkeletonPresets';

if (isLoading) {
    return <SkeletonCard />;
}

    üîê Authentication

    User Roles

    • admin - Full access
    • editor - Create/edit projects
    • viewer - Read-only access
    • user - Basic access

    Protected Routes

    <Route path="/protected" element={
    <ProtectedRoute>
        <Component />
    </ProtectedRoute>
} />

    Check Permissions

    import { checkAccess } from '@/server/lib/auth';

if (!checkAccess(resource, user, 'write')) {
    return res.status(403).json({ error: 'Access denied' });
}

    üìä Analytics Events

    Page Views

    analytics.pageView({ path: '/page', title: 'Page Title' });

    Authentication

    analytics.login(userId, 'email' | 'google' | 'microsoft');
analytics.logout(userId);

    Features

    analytics.featureUsed('Feature Name', { metadata });
analytics.prototypeGenerated({ model, duration, success });

    Errors

    analytics.error(error, { context: 'Context Name' });

# Pull model

# Test model
