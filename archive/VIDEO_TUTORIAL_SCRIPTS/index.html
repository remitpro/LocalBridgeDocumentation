    üìù Production Notes

    Equipment Needed

    • Screen recording software (OBS Studio, Camtasia, ScreenFlow)
    • Microphone (USB condenser mic recommended)
    • Video editing software (Final Cut Pro, Premiere, DaVinci Resolve)

    Recording Tips

    1. Preparation:
    2. Test all steps before recording
    3. Prepare clean demo environment
    4. Close unnecessary applications

    5. Set browser zoom to 100%

    6. During Recording:

    7. Speak clearly and at moderate pace
    8. Pause between sections
    9. Use cursor highlighting

    10. Show keyboard shortcuts on screen

    11. Post-Production:

    12. Add intro/outro animations
    13. Include captions/subtitles
    14. Add background music (subtle)
    15. Export in 1080p minimum

    Branding

    • Use LocalBridge colors consistently
    • Include logo in corner
    • Add end cards with links
    • Maintain professional tone
